## Overview
This file defines the `AgentState` TypedDict for a LangGraph-based agent workflow. It structures the state data that flows between graph nodes during document analysis, including chat history, extracted metadata, detected matters, analysis results, and LaTeX generation control fields.

## Classes
| Name | Primary Duty |
