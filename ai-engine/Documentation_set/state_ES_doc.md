## Descripción general
Este archivo define el `AgentState` TypedDict para un flujo de trabajo de agente basado en LangGraph. Estructura los datos de estado que fluyen entre los nodos del grafo durante el análisis de documentos, incluyendo el historial del chat, metadatos extraídos, asuntos detectados, resultados de análisis y campos de control de generación LaTeX.

## Clases
| Nombre | Deber principal |
|------|--------------|
| `AgentState` | Define el esquema de estado completo para el agente de análisis de documentos, manteniendo el contexto conversacional, datos extraídos, salidas de análisis y control de generación a través de los pasos de ejecución del grafo. |
